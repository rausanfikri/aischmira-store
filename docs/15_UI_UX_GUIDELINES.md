# UI / UX Guidelines

# Philosophy

AISCHMIRA bukan marketplace.

AISCHMIRA adalah Luxury Fashion Brand.

Karena itu website harus terasa seperti memasuki butik premium.

Pengguna harus fokus pada produk, cerita brand, dan pengalaman.

Prioritas:

Elegan

Luxury

Minimal

Editorial

Clean

Timeless

Calm

---

# Design Principles

1.

Whitespace lebih penting daripada memenuhi layar.

2.

Gunakan sedikit warna.

3.

Gunakan animasi seperlunya.

4.

Jangan membuat halaman terlalu ramai.

5.

Foto produk menjadi pusat perhatian.

6.

Typography adalah elemen desain utama.

# Animation

Gunakan Framer Motion.

Durasi

200-400ms

Easing

---

# Icon

Gunakan Lucide React.

Jangan mencampur berbagai library icon.

---

# Images

Gunakan Next Image.

Semua image responsive.

Gunakan lazy loading.

Gunakan placeholder jika gambar belum tersedia.

---

# Accessibility

Gunakan semantic HTML.

Gunakan alt text.

Gunakan keyboard navigation.

Kontras warna minimal WCAG AA.

# Component Rules

Jangan hardcode ukuran.

Gunakan props.

Gunakan variant jika diperlukan.
